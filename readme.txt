Affiliates, Affiliates Pro and Affiliates Enterprise integration with the Captcha plugin.

== Description ==

Activate and the Captcha will appear on the affiliate registration form when the [Captcha](http://wordpress.org/plugins/captcha/) plugin is activated.

Requirements:

- [Affiliates](http://wordpress.org/plugins/affiliates/) or [Affiliates Pro](http://www.itthinx.com/shop/affiliates-pro/) or [Affiliates Enterprise](http://www.itthinx.com/shop/affiliates-enterprise/)
- [Captcha](http://wordpress.org/plugins/captcha/)
